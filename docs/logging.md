## 📖 Overview

This document serves as the master log for the **KW_CRI** (Coupled Thermodynamic Dispersion Relation) automated sensitivity workflow. Every time an experiment is executed via the primary execution script (`Code/CRI_dispersion.sh`), the pipeline automatically appends the exact run metadata to this ledger.

These logs securely track the mathematical investigation of the linear instability and phase speed of the system's normal modes (Slow Mode, Intermediate Mode, and Fast Wave) across various boundary conditions and parameter grids. Maintaining this history is critical for seamlessly cross-referencing auto-generated visualization heatmaps back to the specific Git commit and physics configurations that produced them.
